Chart Folder Name
  • Describes where Bridge will put the chart inside the chart library directory
  • Use "/" to describe subfolders
  • Use "{{ '{tag}' }}" as a placeholder for chart-specific properties
Available tags:
{{ '{name}, {artist}, {album}, {genre}, {year}, {charter}' }}
Examples:
{{ '{artist} - {name} ({charter})' }}
Creates: Queen - Bohemian Rhapsody (Harmonix)
All charts are placed directly in your library folder.
{{ '{artist}/{name} ({charter})' }}
Creates: Queen/Bohemian Rhapsody (Harmonix)
Charts are organized into subfolders by artist name.
{{ '{genre}/{artist}/{name}' }}
Creates: Rock/Queen/Bohemian Rhapsody
Charts are organized into nested folders by genre, then artist.
